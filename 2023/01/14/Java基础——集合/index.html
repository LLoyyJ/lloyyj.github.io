Java基础——集合

Created2023-01-14|Updated2023-01-18|Java
|Word count:333|Reading time:1min|Post View:

集合

由Collection接口（存放单一元素）和Map接口（存放键值对）派生而来

Collection

List, Set, Queue, Map 四者的区别？

  • List: 存储的元素是有序的、可重复的。
  • Set: 存储的元素是无序的、不可重复的。
  • Queue: 按特定的排队规则来确定先后顺序，存储的元素是有序的、可重复的。
  • Map: 使用键值对（key-value）存储，key 不可重复，value可重复，每个键最多映射到一个值。

List

  • ArrayList： Object[] 数组

    • 适用于频繁的查找工作，线程不安全
    • 快速随机的访问，查找方便，指定位置插入的复杂度为O(n-i)
    • 内存空间占用：在结尾预留一定的容量空间

  • Vector：Object[] 数组

    • 线程安全

  • LinkedList： 双向链表

    • 内存空间占用：每一个元素都需消耗更多的空间（存放前驱元素和后继元素）

      双向链表：prev指向前一节点，next指向下一节点

      双向循环链表：双向链表+首尾节点相连

Set

  • HashSet(无序，唯一)：基于 HashMap 实现的，底层采用 HashMap 来保存元素
  • LinkedHashSet：LinkedHashSet 是 HashSet 的子类
  • TreeSet(有序，唯一)：红黑树(自平衡的排序二叉树)

Quene

  • PriorityQueue： Object[] 数组来实现二叉堆
  • ArrayQueue： Object[] 数组 + 双指针

Map

  • HashMap
  • LinkedHashMap
  • Hashtable
  • TreeMap
