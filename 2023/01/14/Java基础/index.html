数组转换为Stream流的两种方式 —— 将数组转换为Stream流来遍历数组

  • Stream.of(array)
    • 当数组的数据类型为基础类型时——返回数组地址（Stream<int[]>）
    • 当数组的数据类型是包装类时——返回数组中的元素（Stream）
      • 因为此时参数是可变长度的参数
      • 此时跳转至Array.stream()方法中
  • Arrays.stream(array)
    • 当数组的数据类型为基础类型时——返回数组中的元素（IntStream）
    • 当数组的数据类型是包装类时——返回数组中的元素（Stream）

JVM、JRE、JDK

  • JVM：运行字节码文件（.class文件）

  • JRE：运行已编译Java程序 （包括JVM）

  • JDK：Java开发使用，也可编译执行 （包括JRE）

    Java是“编译与解释并存”的语言

    .java文件由编译器编译成.class文件，再由解释器解释为机器码执行。

成员变量&局部变量

  • 语法形式

    访问控制符 static final
    成员变量 √ √ √
    局部变量 × × √

  • 存储方式

    • 成员变量使用static修饰时：属于类
    • 成员变量不使用static修饰时：属于实例
    • 对象存储在堆内存
    • 局部变量存储在栈内存

  • 生存时间

    • 成员变量：随对象创建而存在
    • 局部变量：方法调用时自动生成，方法调用结束后消亡

  • 默认值

    • 成员变量：自动赋以类型的默认值（final修饰的成员变量需要显式地赋值）
    • 局部变量：不会自动赋值

静态变量

​ ——由类的所有实例共享，通常被final修饰为常量使用。

字符型常量&字符串常量

  • 字符型常量：一个整型值（ASCII码），参与表达式计算（Java中占两个字节）
  • 字符串常量：代表一个地址值

静态方法&实例方法

  • 调用方式

    • 静态方法——类名.方法名（调用静态方法无需创建对象）（一般不建议使用对象.方法名调用静态方法以免混淆）
    • 实例方法——对象.方法名

  • 访问类成员

    • 静态方法：仅可访问静态成员（即静态成员变量和静态方法）
    • 实例方法：可以访问静态成员&实例成员
    • 类的主方法：可以直接调用静态成员方法；可以通过对象调用实例成员方法

重载&重写

  • 重载（编译器） —— 同一个类中多个同名方法根据不同地产参来执行不同的逻辑处理
    • 发生在同一个类当中 / 父类与子类之间
    • 方法名必须相同；参数类型、个数、顺序不同；返回值和访问修饰符可以不同
    • 编译器挑选方法时，如果找不到匹配的参数就会产生编译时错误（这个过程称为重载解析（overloading resolution））
  • 重写（运行期）—— 子类对父类方法的重新改造，外部样子不能改变，内部逻辑可以改变
    • 方法名、参数列表必须相同
    • 限制：
      • 子类方法返回值应 <= 父类方法返回值类型
      • 子类抛出的异常范围 <= 父类抛出的异常范围
      • 子类的访问修饰符范围 >= 父类的访问修饰符范围
    • 访问修饰符为 private/final/static的父类方法不可以被重写，但是被static修饰的方法能够被重新声明
    • 构造方法无法被重写
    • 返回值类型：
      • void / 基本数据类型 —— 重写是返回值不可修改
      • 引用类型 —— 可以返回引用类型的子类
